# Playwright Python Automation Project

This project is set up for browser automation testing using Playwright and pytest.

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   playwright install
   ```
2. Run tests:
   ```bash
   pytest
   ```

## Project Structure
- `tests/` - Example Playwright tests
- `requirements.txt` - Python dependencies
- `pytest.ini` - Pytest configuration
